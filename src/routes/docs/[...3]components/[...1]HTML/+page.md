# HTML

A Svelte html component to wrap emails.

## Usage

```svelte
<script>
	import { Html } from 'svelte-mail';
</script>

<Html lang="en">
	<!-- ... -->
</Html>
```

## Props

<script>
	import { Chip } from '@svelteness/kit-docs';
</script>

| Name              | Type     | Description                |
| ----------------- | -------- | -------------------------- |
| <Chip>lang</Chip> | `string` | The language of the email. |
